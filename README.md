# Hexagon Pioneers
## Running the Project
Download node and npm<br/>
In project directory, run `npm install` from both ui and backend folder.<br/>
Next, run `npm run backend` then `npm run frontend` from separate shells.<br/>
To run test suite, run `npm test`
